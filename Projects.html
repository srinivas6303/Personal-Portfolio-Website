Hospital Management System

Technologies: Java, SQL, JDBC | IDE: IntelliJ IDEA

Overview: The Hospital Management System is designed to automate and manage daily hospital operations. It manages information about patients, doctors, appointments, and medical records. The system was created with a focus on improving workflow efficiency and patient experience.

  • Patient, Doctor, and Appointment management using separate classes.

  • Driver class with menu for database interactions and a smooth user interface.

  • Implemented various CRUD operations for managing patient records, doctor assignments, and appointment scheduling.

  • Relational database design using MySQL to store and manage data efficiently.

  • Developed with a modular structure for easy scalability and maintenance.

Challenges & Solutions: One of the challenges was optimizing database queries to handle large datasets of patients. By using indexed queries and limiting the results shown per page, the system became more responsive and efficient.

StudentInfo Management REST API

Technologies: Spring Boot, JDBC | IDE: Eclipse

Overview: The StudentInfo Management REST API is a backend system designed to manage student records at an educational institution. It provides a set of CRUD operations to allow administrators to add, update, and delete student data, and retrieve student information.

  • CRUD operations on MySQL database using Spring Data JPA.

  • API tested with Postman and deployed on Tomcat for real-time functionality testing.

  • Secure authentication mechanisms were implemented for data security, including JWT-based token authentication for API requests.

  • Provides endpoints for creating, reading, updating, and deleting student records.

Challenges & Solutions: One challenge was managing multiple simultaneous requests to the API. This was solved by optimizing the database queries, ensuring efficient handling of concurrent user access through Spring Boot's asynchronous features.

Alien Management REST API

Technologies: Jersey, JDBC | IDE: Eclipse

Overview: The Alien Management REST API is an application designed to manage data about extraterrestrial entities. It allows users to store and retrieve information on different alien species, their characteristics, and encounters.

  • Implemented CRUD operations with JDBC for managing alien records stored in a MySQL database.

  • Supports both JSON and XML data formats for flexibility in data exchange.

  • API endpoints designed to provide efficient data retrieval with multiple search and filter capabilities.

  • Tested thoroughly with Postman to ensure the API was functioning as expected, with unit tests added to validate business logic.

Challenges & Solutions: One significant challenge was ensuring that the API could handle both JSON and XML responses efficiently. The solution was to implement content negotiation based on the request header, allowing the client to choose the format based on their preference.
